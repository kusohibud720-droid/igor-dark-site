Где живёт этот генератор?

Сейчас — это файл на компьютере (как ярлык на рабочем столе). Открыл, сделал пост, опубликовал. Интернет нужен только в момент публикации.

Вариант 1: Файл на компьютере

Просто открываешь этот файл в браузере. Работает без интернета (кроме момента публикации). Ничего устанавливать не нужно. Бесплатно.

Вариант 2: Приложение на телефоне (рекомендую)

Добавляем генератор на сайт (скрытая страница, только для тебя). Потом на телефоне:

Android: открой в Chrome → три точки ⋮ → «Добавить на главный экран»

iPhone: открой в Safari → кнопка «Поделиться» ↑ → «На экран Домой»

На экране появится иконка «ИМ» — как обычное приложение. Открывается в полный экран, без адресной строки. Никакого App Store, никакой платы.

Вариант 3: Telegram-бот (на будущее)

Пишешь боту в Telegram: «Отзыв, свадьба, Павел и Алина, цитата...» — бот собирает пост и публикует. Самый быстрый способ, но нужно настроить бота (могу сделать).

Всё бесплатно. Сайт на бесплатном хостинге, API VK и Telegram бесплатные, App Store не нужен.
